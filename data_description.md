# Описание признаков ГИС

## Геометрические параметры
- **СКВАЖИНА** - идентификатор скважины
- **ГЛУБИНА_ИЗМЕРЕННАЯ** - измеренная глубина (м)
- **КООРДИНАТА_X, Y, Z** - пространственные координаты

## Литологические параметры  
- **ГРУППА** - геологическая группа
- **ФОРМАЦИЯ** - название формации
- **ЛИТОЛОГИЯ** - целевая переменная (тип горной породы)

## Стандартный каротажный комплекс

### Гамма-каротаж (ГК)
- **ГК** - естественная гамма-радиоактивность (API)
- **ГК_СПЕКТРАЛЬНЫЙ** - спектральный гамма-каротаж

### Электрический каротаж (RES)
- **RES_ЗОНА_ВТОРЖЕНИЯ** - сопротивление зоны проникновения (Ом·м)
- **RES_СРЕДНЯЯ** - среднее сопротивление (Ом·м) 
- **RES_ГЛУБОКАЯ** - глубокое сопротивление (Ом·м)
- **RES_МИКРО** - микрокаротаж (Ом·м)

### Акустический каротаж
- **DTC** - интервальное время пробега (мкс/фут)
- **DTS** - время пробега поперечной волны

### Нейтронный каротаж
- **ННК** - нейтронный пористость (д.ед.)

### Плотностной каротаж  
- **ПЛОТНОСТЬ** - объемная плотность (г/см³)
- **ПЛОТНОСТЬ_ПОПРАВКА** - поправка к плотности

### Дополнительные измерения
- **КАЛИБР** - диаметр скважины (дюймы)
- **КАЛИБР_РАСХОЖДЕНИЕ** - расхождение калибра
- **СП** - спонтанная поляризация (мВ)
- **PEF** - фотoelectric factor (б/н)

## Технологические параметры
- **РАЗМЕР_ДОЛОТА** - диаметр долота (дюймы)
- **СКОРОСТЬ_ПРОХОДКИ** - скорость бурения (м/час)
- **ПЛОТНОСТЬ_РАСТВОРА** - плотность бурового раствора (г/см³)
